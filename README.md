# ssh4windows10  
**Version:** 1.0.1

## Description  
Easy tool to install and run SSH on Windows.  
With this tool you can install and run SSH using CMD on Windows.

## Requirements  
- Windows 10 Pro version 22H2  
- Must run as Administrator

## Download  
[https://github.com/RedDarkdevXD/ssh4Windows10‑BatScript.git](https://github.com/RedDarkdevXD/ssh4Windows10‑BatScript.git)

## Usage  
1. Run CMD as Administrator.  
2. Browse to the folder containing the script.  
3. Execute the script.  
4. Use the menu to select an option:  
   - `0` — Exit  
   - `1` — Instalar SSH  
   - `2` — Iniciar SSH  
   Simply type the option number and press Enter. The script will do the rest.
